## Steps to run the app

```bash
# Remove the "target" directory
$ mvn clean

# Compile the project source code
$ mvn compile

# Compile and package your application into a JAR file
$ mvn package

# Ensure dependencies are resolved correctly
$ mvn dependency:resolve

# Run tests
$ mvn test

# Start the application
$ mvn spring-boot:run

# Java Version
Use Java 17 - compatible with the latest Spring Boot versions

## Used Libraries
- H2: In-memory database
- Lombok: Simplifies code by avoiding getters, setters, and more
- JUnit 4: Testing framework
- Swagger: UI for testing the endpoints

## Swagger UI
Access Swagger UI at: http://localhost:8080/doc/swagger-ui/index.html
